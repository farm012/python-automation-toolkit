# Bulk File Renamer

A simple Python tool that helps you rename multiple files quickly and consistently.

## The Problem

Renaming hundreds of files manually is slow and annoying.

This tool helps you:
- Rename many files at once
- Keep files organized
- Avoid repetitive manual work

## Features

- Bulk file renaming
- Simple command-line interface
- Fast processing
- Lightweight Python script
